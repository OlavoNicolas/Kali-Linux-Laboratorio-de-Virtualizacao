## Kali Linux: Otimização de Laboratório de Virtualização
Este repositório documenta a arquitetura e implementação de um laboratório de segurança ofensiva com Kali Linux, focado em instalação mínima (barebone) e alta performance para ambientes com hardware limitado.

## Objetivo
Demonstrar competências em administração de sistemas Linux e virtualização, criando um ambiente controlado e otimizado para estudos de segurança e redes.

## Especificações do Ambiente
Para este projeto, optei por uma configuração de baixo consumo de recursos, mantendo a estabilidade do sistema:
- Hypervisor: VirtualBox
- RAM: 2 GB
- CPU: 1 núcleo
- Disco: 30 GB
- Rede: NAT
- Desktop: XFCE

## Etapas realizadas
- Criação da máquina virtual
- Instalação mínima do Kali Linux
- Configuração de desktop leve (XFCE)
- Otimização de serviços
- Validação do consumo de recursos

## Instalação
- Installer: Graphical interface
- Desktop: XFCE
- Pacotes: mínimos (sem default tools)

## Pós-instalação
- Atualização do sistema
- Desativação de serviços desnecessários

## ⚠️ Observações
Este laboratório foi criado exclusivamente para fins educacionais,
em ambiente controlado.

## Setup do Laboratório Kali Linux
### Criação da VM
- RAM: 2048 MB
- CPU: 1 core
- Vídeo: 128 MB
- Rede: NAT

## Resultados Obtidos
- Tempo de Boot: Redução significativa comparada à instalação padrão.
- Estabilidade: Ambiente fluido mesmo com múltiplas abas de terminal e ferramentas de análise abertas.
